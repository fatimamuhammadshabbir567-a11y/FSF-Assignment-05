# 🎓 FSF Assignment 05 - Grade Calculator

A modern and responsive **Grade Calculator** built using **HTML, CSS, and JavaScript**. This project calculates a student's grade based on the entered percentage using **if...else statements** and includes input validation with a clean and attractive user interface.

---

## 📌 Project Overview

This Grade Calculator allows users to enter their percentage and instantly view their grade. The application validates user input and displays appropriate messages for invalid or empty input.

---

## ✨ Features

- 🎯 Calculate grades based on percentage
- ✅ Uses JavaScript **if...else** statements
- 🚫 Invalid number validation (Below 0 or Above 100)
- ⚠ Empty input validation
- 🎨 Modern Glassmorphism UI
- 📱 Fully Responsive Design
- 🌈 Attractive Gradient Background
- ✨ Smooth Hover Effects
- 💻 Works on Desktop, Tablet, and Mobile devices

## 📊 Grading Criteria

| 70 – 79 | B |
| 60 – 69 | C |
| 40 – 59 | D |
